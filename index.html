Política de Privacidade

Última atualização: 2026

O aplicativo AutomBot Agenda Pro respeita a privacidade dos usuários e se compromete a proteger qualquer informação coletada durante o uso do aplicativo.

1. Coleta de Informações

O AutomBot Agenda Pro não coleta, armazena ou compartilha dados pessoais diretamente dos usuários.

No entanto, o aplicativo pode utilizar serviços de terceiros, como o Google AdMob, que podem coletar informações automaticamente para exibição de anúncios.

  • Dados de uso do dispositivo
  • Identificadores anônimos
  • Interações com anúncios

2. Uso das Informações

As informações coletadas automaticamente são utilizadas para:

  • Exibir anúncios relevantes
  • Melhorar a experiência do usuário
  • Analisar o desempenho do aplicativo

3. Compartilhamento de Dados

O aplicativo não compartilha dados pessoais diretamente. Dados coletados por terceiros seguem suas próprias políticas.

4. Serviços de Terceiros

Este aplicativo pode utilizar:

  • Google AdMob – https://policies.google.com/privacy

5. Segurança

São adotadas boas práticas para garantir a segurança da aplicação.

6. Direitos do Usuário

Como não há coleta direta de dados pessoais, não há armazenamento para consulta, correção ou exclusão.

7. Alterações

Esta política pode ser atualizada a qualquer momento. Recomendamos revisão periódica.
